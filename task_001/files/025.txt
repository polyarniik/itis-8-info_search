<b>Почему производителям не выгодно делать надежную офисную технику?</b><br/>
<br/>
<h2><font color="#1c22be">❯</font> <font color="#454cee">Предыстория</font></h2><br/>
Все началось в 1924 году, когда производители ламп, среди которых были <a href="https://ru.wikipedia.org/wiki/Osram">Osram</a>, <a href="https://ru.wikipedia.org/wiki/Philips">Philips</a>, <a href="https://ru.wikipedia.org/wiki/General_Electric">General Electric</a>, организовали <a href="https://ru.wikipedia.org/wiki/%D0%9A%D0%B0%D1%80%D1%82%D0%B5%D0%BB%D1%8C_Phoebus">картель Phoebus («Феб»)</a> и договорились искусственно ограничить срок службы лампочек за счет снижения качества применяемых материалов. Сговор раскрылся в 1942 году.<br/>
<br/>
<div class="spoiler" role="button" tabindex="0">
<b class="spoiler_title">«Столетняя лампа», которая занесена в Книгу рекордов Гиннеса</b>
<div class="spoiler_text"><div style="text-align:center;"><img data-src="https://habrastorage.org/webt/is/tb/69/istb69cz3vbuymy1njj-dqq6as4.png" src="https://habrastorage.org/r/w1560/webt/is/tb/69/istb69cz3vbuymy1njj-dqq6as4.png"/></div><br/>
Лампа накаливания, горящая с 1901 года, город Ливермор, Калифорния.<br/>
<br/>
<a href="https://ru.wikipedia.org/wiki/%D0%A1%D1%82%D0%BE%D0%BB%D0%B5%D1%82%D0%BD%D1%8F%D1%8F_%D0%BB%D0%B0%D0%BC%D0%BF%D0%B0">Источник</a> <br/>
<br/>
</div>
</div> <br/>
Первым описал идею «<a href="https://ru.wikipedia.org/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D0%BE%D0%B5_%D1%83%D1%81%D1%82%D0%B0%D1%80%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5">Запланированного старения</a>» американец <a href="https://translated.turbopages.org/proxy_u/en-ru.ru.4483b12c-63f0d52c-9d78fc9d-74722d776562/https/en.wikipedia.org/wiki/Bernard_London">Бернард Лондон</a> в своей книге «Конец депрессии через планируемое устаревание», 1932 год.<br/>
<br/>
Идея описанная в книге проста. Товар с коротким сроком использования побуждает потребителя совершать повторную покупку. <br/>
<br/>
<b>Можно выделить два способа ограничения срока эксплуатации:</b><br/>
<br/>
<ul>
<li>Техническое ограничение, товар ломается в запланированные сроки, прекращается техническая поддержка или выпуск комплектующих и расходных материалов.</li>
<li>Маркетинговые инструменты*, товар делают не модным, самый яркий пример: смартфон «iphone».</li>
</ul><br/>
<div class="spoiler" role="button" tabindex="0">
<b class="spoiler_title">* Книга «Система вещей» французского социолога и философа Жана Бодрийяра.</b>

<br/>
</div>
</div><br/>
В 2022 году появился еще один инструмент ограничения эксплуатации — «санкции».<br/>
<br/>
<div class="spoiler" role="button" tabindex="0">
<b class="spoiler_title">Уход из России крупнейших производителей принтеров привёл к переделу рынка.</b>
<div class="spoiler_text">В 2021 году Российский рынок принтеров HP vs Pantum:<br/>
<br/>
<ul>
<li>HP (США) — 36%</li>
<li>Pantum (Китай) — 16% </li>
</ul><br/>
В 2022 году Российский рынок принтеров изменился:<br/>
<br/>
<ul>
<li>Pantum (Китай) — 67% </li>
<li>HP (США) — 12%</li>
</ul><br/>
</div>
</div><br/>
<h2><font color="#1c22be">❯</font> <font color="#454cee">Принтеры</font></h2><br/>
1843 год — появление идеи принтера у <a href="https://ru.wikipedia.org/wiki/%D0%91%D1%8D%D0%B1%D0%B1%D0%B8%D0%B4%D0%B6,_%D0%A7%D0%B0%D1%80%D0%BB%D0%B7">Чарлза Бэббиджа</a>, только спустя 150 лет по его чертежам был воспроизведен принтер, весящий 5 тонн.<br/>
<br/>
1964 год — первый матричный принтер, компания <a href="https://ru.wikipedia.org/wiki/Seiko_Epson">Seiko Epson Corporation</a> впервые реализовала идею матричной печати в устройстве.<br/>
<br/>
<div style="text-align:center;"><img data-blurred="true" data-src="https://habrastorage.org/webt/l8/66/0o/l8660objtdcb30yldoqg2bzlcbi.jpeg" src="https://habrastorage.org/r/w780q1/webt/l8/66/0o/l8660objtdcb30yldoqg2bzlcbi.jpeg"/></div><i><font color="#4682B4">Принтер EP-101</font></i><br/>
<br/>
1978 год — первый лазерный принтер, полнодуплексный <a href="https://en.wikipedia.org/wiki/Xerox_9700">Xerox 9700</a> печатал 120 страниц в минуту. В 2023 году Xerox 9700 остается одним из быстрых принтеров в мире. <br/>
<br/>
<div style="text-align:center;"><img data-blurred="true" data-src="https://habrastorage.org/webt/99/zi/7h/99zi7hyv1qf01tf-12gwdhuvmtq.jpeg" src="https://habrastorage.org/r/w780q1/webt/99/zi/7h/99zi7hyv1qf01tf-12gwdhuvmtq.jpeg"/></div><br/>
В 70-х годах цена Xerox 9700 — 350 000 USD.<br/>
<br/>

<h2><font color="#1c22be">❯</font> <font color="#454cee">Мои принтеры</font></h2><br/>
Выбирая для работы принтер опирался на два параметра — это ресурс и ремонтопригодность.<br/>
<br/>

<br/>
Был куплен 2007 году.<br/>
<br/>

<br/>
Слабое место этого МФУ — сканер, для дома справляется с любыми задачами, в офисе при большом объеме сканирования долго не протянет.<br/>
<br/>
Отличительная черта этой модели — хорошая ремонтопригодность. <br/>

<b>В офисе использую лазерный принтер KYOCERA ECOSYS P2135dn.</b><br/>
<br/>
Один из самых надежных принтеров, который я знаю. Давно завел правило — «подменный фонд принтеров». Это позволяет не останавливать рабочие процессы в компании.<br/>
<br/>
Вот так хранятся подменные принтеры.<br/>
<br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/ns/2r/zl/ns2rzlj5rk5hijswlhexczmh4oc.jpeg" src="https://habrastorage.org/r/w780q1/webt/ns/2r/zl/ns2rzlj5rk5hijswlhexczmh4oc.jpeg"/><br/>
<br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/9r/8v/nj/9r8vnjcv4dwzex_9fya2gxcj_94.jpeg" src="https://habrastorage.org/r/w780q1/webt/9r/8v/nj/9r8vnjcv4dwzex_9fya2gxcj_94.jpeg"/><br/>
<br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/v2/-a/mw/v2-amwziimqgxsfue7g1qa5ioo8.jpeg" src="https://habrastorage.org/r/w780q1/webt/v2/-a/mw/v2-amwziimqgxsfue7g1qa5ioo8.jpeg"/><br/>
<br/>
Пример, как легко ремонтируется ECOSYS P2135dn.<br/>
<br/>
<div class="oembed"><div class="tm-iframe_temp" data-src="https://embedd.srv.habr.com/iframe/63e9576f7a8d27e0260eb9f5" data-style="" id="63e9576f7a8d27e0260eb9f5" width=""></div></div> <br/>
В принтере используется раздельный барабан на 100 000 страниц печати.<br/>
Барабан ремонтопригодный, за исключением случаев физического разрушения. <br/>
<br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/hg/el/z2/hgelz2et-dahm-atf2b2kxmv1hc.jpeg" src="https://habrastorage.org/r/w780q1/webt/hg/el/z2/hgelz2et-dahm-atf2b2kxmv1hc.jpeg"/><br/>
<br/>
<div class="spoiler" role="button" tabindex="0">
<b class="spoiler_title">Важные характеристики</b>
<div class="spoiler_text">Картридж на 7200 страниц печати.<br/>
Входит 290 гр. тонера, цена печати 10 копеек.<br/>
<br/>
Тип печати — черно-белая <br/>
Плотность бумаги — 60-220 г/м2<br/>
Объем памяти — 32 Мб, максимальный 512 Мб<br/>
Максимальный формат — A4 <br/>
Максимальное разрешение для ч/б печати — 1200x1200 dpi <br/>
Время выхода первого отпечатка — 8 c <br/>
Скорость печати — 35 стр/мин <br/>
<br/>
Интерфейсы — Ethernet (RJ-45), USB 2.0 <br/>
ОС — Windows, Linux, Mac OS <br/>
Потребляемая мощность (при работе) — 535 Вт<br/>
Потребляемая мощность (в режиме ожидания) — 11 Вт<br/>
Габариты (ШхВхГ) — 375x267x393 мм<br/>
Вес — 12 кг<br/>
</div>
</div><br/>
<b>Если сравнить KYOCERA ECOSYS P2135dn и KYOCERA ECOSYS P2235dn, то вы увидите, что изменил производитель в новой модели!</b><br/>
<br/>
<img data-src="https://habrastorage.org/webt/_1/rf/3r/_1rf3rbhwo0xbjzr989kzloteta.png" src="https://habrastorage.org/r/w1560/webt/_1/rf/3r/_1rf3rbhwo0xbjzr989kzloteta.png"/><br/>
<br/>
Производитель увеличил стоимость новой модели и уменьшил количество тонера, в результате картриджа хватает на 3000 страниц печати. <br/>
<br/>
Чтобы еще уменьшить срок службы, производитель <a href="https://ru.wikipedia.org/wiki/Kyocera">KYOCERA</a> изменил устройство термоузла. <br/>
В модели ECOSYS P2135dn используется тефлоновый вал с ресурсом 100 000 страниц, как он легко меняется, показано на видео выше. Для уменьшения срока службы принтера в модели ECOSYS P2235dn тефлоновый вал заменили термопленкой, ресурс всего термоузла (печка) уменьшился до 70 000 страниц. Первый раз возможно отремонтировать печку ECOSYS P2235dn, но в следующий ремонт потребуется полностью заменить термоузел (печку). <br/>
<br/>
<div class="oembed"><div class="tm-iframe_temp" data-src="https://embedd.srv.habr.com/iframe/63e9f22261f0ac6fb260fdeb" data-style="" id="63e9f22261f0ac6fb260fdeb" width=""></div></div><br/>
Производство пластиковых деталей не стоит на месте, можно запланировать изменение свойств, и через несколько лет под нагрузкой начинает разрушаться копеечная шестеренка. Проблема на 5 копеек, а принтер становится не пригоден для эксплуатации. <br/>
<br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/au/s2/rk/aus2rkdxrjqt--eazvwjrlnb1dm.jpeg" src="https://habrastorage.org/r/w780q1/webt/au/s2/rk/aus2rkdxrjqt--eazvwjrlnb1dm.jpeg"/><br/>
<br/>
Или использование в новой технике бескорпусных микросхем-«капелька». Микросхему-капельку практически невозможно перепаять.<br/>
<br/>
<img data-src="https://habrastorage.org/webt/vg/ps/o0/vgpso0qaz-thpi6hattqp7g8clg.png" src="https://habrastorage.org/r/w1560/webt/vg/ps/o0/vgpso0qaz-thpi6hattqp7g8clg.png"/> <br/>
<br/>
Хорошо, что пока с такими проблемами в KYOCERA не сталкивался и это радует.<br/>
<br/>
Для себя решил следующее, пока можно купить живые б/у KYOCERA ECOSYS P2135dn, а также пока есть ремкомплекты на надежные старые модели, на новые принтеры переходить не буду.<br/>
<br/>
Сейчас принтер KYOCERA ECOSYS P2135dn б/у в хорошем состоянии можно купить от 10 000 до 12 000 р. <br/>
<br/>
P. S.Старый лазерный «друг» KYOCERA ECOSYS P2135dn, лучше новых двух KYOCERA ECOSYS P2235dn.<br/>
<br/>
Благодарю, что дочитали.<br/>
