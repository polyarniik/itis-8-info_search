article-formatted-body_version-1"><div xmlns="http://www.w3.org/1999/xhtml"><a href="https://habr.com/ru/company/ruvds/news/t/717466/"><div style="text-align:center;"><img data-blurred="true" data-src="https://habrastorage.org/webt/nl/ff/9q/nlff9qpt_4_tsm-txktrwdddw_w.jpeg" src="https://habrastorage.org/r/w780q1/webt/nl/ff/9q/nlff9qpt_4_tsm-txktrwdddw_w.jpeg"/></div></a><br/>
На днях Slack объявил, что в марте 2023 года полностью прекращает деловые отношения с клиентами из России. Компания планирует приостановить любой доступ к рабочим пространствам россиян 21 марта 2023 года и удалить все имеющиеся там данные в течение следующих 60 дней. Также ранее корпорация Майкрософт <a href="https://learn.microsoft.com/ru-ru/microsoftteams/teams-add-on-licensing/remove-assign-free-licenses">заявила</a> о прекращении использования лицензии на бесплатную версию Microsoft Teams с середины апреля 2023 года. <br/>
<br/>
В связи с этим сейчас многие компании срочно ищут замену в виде более удобного, дешёвого и надёжного варианта для корпоративных коммуникаций. RUVDS предлагает своим клиентам <a href="https://ruvds.com/ru-rub/marketplace/matrix-synapse#order">решение</a> на базе опенсорсного протокола Matrix.<a name="habracut"></a><br/>
<br/>
<h2><font color="#3AC1EF">▍ Протокол Matrix</font></h2><br/>
Matrix представляет собой набор API (JSON over REST), который даёт возможность мгновенно обмениваться сообщениями, поддерживает обмен файлами, IP-телефонию, видеосвязь, IoT и др. Он не только передаёт данные между клиентом и сервером, но и организует связь между серверами благодаря децентрализованному механизму <a href="https://ru.wikipedia.org/wiki/Fediverse">федерации</a>. <br/>
<br/>
<div style="text-align:center;"><img data-blurred="true" data-src="https://habrastorage.org/webt/j1/z4/ua/j1z4uayrmoofslewdyqckkcutx0.jpeg" src="https://habrastorage.org/r/w780q1/webt/j1/z4/ua/j1z4uayrmoofslewdyqckkcutx0.jpeg" width="550"/></div><br/>
<i><font color="#999999">Чёрным цветом на изображении помечены серверы Matrix, зелёным — клиенты Element, синим — мосты в другие сети</font></i><br/>
<br/>
На всякий случай напомним основные преимущества децентрализации: высокая безопасность, надёжность и устойчивость сети, полное владение собственными данными, свобода и независимость. Любая компания может создать собственную «федерацию» для внутрикорпоративного взаимодействия, подняв сервер и подключившись к глобальной сети Matrix. <br/>
<br/>
<h2><font color="#3AC1EF">▍ Element — клиент для Matrix</font></h2><br/>
В настоящее время широкой популярностью пользуется клиент Element, который многие даже называют официальным клиентом Matrix. Среди его преимуществ стоит отметить:<br/>
<br/>
<ul>
<li>сквозное шифрование по умолчанию;</li>
<li>хранение данных на собственном сервере (on-premise) или Element Cloud в любой точке мира;</li>
<li>приложения для Android, iOS, Windows, macOS и Linux, а также работа в веб-браузере;</li>
<li>мосты в другие платформы для общения (Signal, Telegram, Facebook Messenger, Google Hangouts, Skype, Discord и т. д.);</li>
<li>умная верификация новых устройств через QR-код или последовательность эмодзи;</li>
<li>возможность проведения видеоконференций, что делает его достойной заменой не только Slack и Microsoft Teams, но также Zoom и Skype for Business.</li>
</ul><br/>
<h2><font color="#3AC1EF">▍ Сервер MATRIX SYNAPSE</font></h2><br/>
В маркетплейсе можно в пару кликов <a href="https://ruvds.com/ru-rub/marketplace/matrix-synapse#order">бесплатно протестировать</a> виртуальный сервер с уже настроенным клиентом Element, поддерживающим по умолчанию сквозное шифрование и хранение данных на своём сервере. Платить за использование программного обеспечения не нужно, затраты ограничатся стоимостью виртуального сервера (рекомендуемая конфигурация на 30 пользователей обойдётся в 899 рублей в месяц). В образе MATRIX SYNAPSE уже есть небольшая открытая утилита Synapse admin UI с веб-интерфейсом для администрирования, а также веб-версия клиента — Element Web.<br/>
<br/>
<div style="text-align:center;"><img data-src="https://habrastorage.org/webt/17/a0/bu/17a0bus5tg6n6s7h1v9lht5hth4.png" src="https://habrastorage.org/r/w1560/webt/17/a0/bu/17a0bus5tg6n6s7h1v9lht5hth4.png"/></div><br/>
<br/>
<h2><font color="#3AC1EF">▍ Подведём итоги</font></h2><br/>
Комбинация Element/Matrix выигрывает у Slack по всем параметрам:<br/>
<ul>
<li>стоимость (недорогой виртуальный сервер против корпоративного тарифа Slack);</li>
<li>более надёжное шифрование и высокая безопасность;</li>
<li>контроль за инфраструктурой (вы сами управляете коммуникациями, никакая информация не уходит во внешний мир);</li>
<li>широкий набор функций, в том числе в Matrix API:<br/>
<br/>
<ol>
<li><a href="https://spec.matrix.org/v1.5/client-server-api/">Client-Server API</a></li>
<li><a href="https://spec.matrix.org/v1.5/server-server-api/">Server-Server API</a></li>
<li><a href="https://spec.matrix.org/v1.5/application-service-api/">Application Service API</a></li>
<li><a href="https://spec.matrix.org/v1.5/identity-service-api/">Identity Service API</a></li>
<li><a href="https://spec.matrix.org/v1.5/push-gateway-api/">Push Gateway API</a></li>
</ol></li>
</ul><br/>
Если рассматривать мировые тренды в области корпоративных мессенджеров, то в настоящий момент централизованные приложения (Slack, Teams, Discord и т. д.) теряют свои позиции, а на первые места выходят решения с открытым исходным кодом. Это напоминает нам то, как Linux вытеснил коммерческие ОС и в целом повысил популярность опенсорса. <br/>
