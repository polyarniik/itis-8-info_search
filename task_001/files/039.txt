Не менее интересно, что DevTools получил новые инструменты, помогающие разработчикам создавать, конвертировать и отлаживать цвета высокой четкости.</p><p>Панель стилей теперь поддерживает 12 новых цветовых пространств и 7 новых гамм, как описано в спецификации <a href="https://www.w3.org/TR/css-color-4/" rel="noopener noreferrer nofollow">CSS Color Level 4</a>. Смотрите <a href="https://developer.chrome.com/articles/high-definition-css-color-guide/#debugging-color-with-chrome-devtools" rel="noopener noreferrer nofollow">Руководство по цвету CSS высокой четкости</a> для получения полного представления о вариантах цвета.</p><p>Вот примеры использования CSS стилей с помощью <code>color()</code>, <code>lch()</code>, <code>oklab()</code> и <code>color-mix()</code>.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/75a/2cc/777/75a2cc7778ce7bde7ac662bc79324bfc.png" height="1178" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/75a/2cc/777/75a2cc7778ce7bde7ac662bc79324bfc.png" width="1850"><figcaption></figcaption></img></figure><p>При использовании функции color-mix() вы можете просмотреть конечный результат цветопередачи на вычисляемой панели.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/fb9/427/7f4/fb94277f4825669bd615ddfd0c5f5951.png" height="1286" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/fb9/427/7f4/fb94277f4825669bd615ddfd0c5f5951.png" width="2114"/><figcaption></figcaption></figure><p>Средство выбора цвета поддерживает все новые цветовые пространства с большим количеством функций. </p><p>Например, щелкните по цветовому образцу <code>color(display-03 1 0 1)</code>. Также была добавлена линия границы гаммы, различающая <code>sRGB</code> и <code>display-p3</code> для лучшего понимания гаммы выбранного вами цвета.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/682/492/534/68249253475bd2dc8423123db48c7cc5.png" height="1008" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/682/492/534/68249253475bd2dc8423123db48c7cc5.png" width="1228"/><figcaption></figcaption></figure><p>DevTools поддерживает преобразование цветов между цветовыми форматами. Используйте значок "Изменить формат цвета", чтобы открыть всплывающее окно преобразования, или просто нажмите <code>Shift</code> + <code>клик</code> на образец цвета на панели стилей.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/cbd/a53/b59/cbda53b59f86c50c1fbe4be82a971813.png" height="1108" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/cbd/a53/b59/cbda53b59f86c50c1fbe4be82a971813.png" width="1928"/><figcaption></figcaption></figure><p>При преобразовании важно знать, было ли преобразование <code>обрезано</code>, чтобы соответствовать <a href="https://ru.wikipedia.org/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BE%D0%B2%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D1%81%D1%82%D1%80%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" rel="noopener noreferrer nofollow">пространству</a> (вики). DevTools предупреждает вас об этом вырезании.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/35e/a2b/320/35ea2b320553640655f84b24376afaa3.png" height="1008" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/35e/a2b/320/35ea2b320553640655f84b24376afaa3.png" width="1228"/><figcaption></figcaption></figure><p>Кроме того, вы можете выбирать цвета со своего экрана с помощью нового ярлыка. Нажмите <code>C</code>, чтобы активировать пипетку, и <code>Escape</code>, чтобы деактивировать ее.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/026/8bf/49c/0268bf49c03d9c3c011d77ac70adff9f.png" height="1008" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/026/8bf/49c/0268bf49c03d9c3c011d77ac70adff9f.png" width="1228"/><figcaption></figcaption></figure><p>Наконец, пункт <code>Color format</code>  теперь устарел, чтобы освободить место для нового цветового формата HD.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/5f0/44e/58d/5f044e58d7824e6e99727337c39a9c95.png" height="732" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/5f0/44e/58d/5f044e58d7824e6e99727337c39a9c95.png" width="1328"/><figcaption></figcaption></figure><p> Известные проблемы: <a rel="noopener noreferrer nofollow">1408777</a>, <a href="https://crbug.com/1395782" rel="noopener noreferrer nofollow">1395782</a>, <a href="https://crbug.com/1392717" rel="noopener noreferrer (<a href="https://bugs.chromium.org/p/chromium/issues/detail?id=508270" rel="noopener noreferrer nofollow">508270</a>, <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=1403362" rel="noopener noreferrer nofollow">1403362</a>, <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=1403432" rel="noopener noreferrer nofollow">1403432</a>, <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=1396298" rel="noopener noreferrer nofollow">1396298</a>, <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=1395337" rel="noopener noreferrer nofollow">1395337</a>, <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=1405134" rel="noopener noreferrer nofollow">1405134</a>)</p></li><li><p>Чтобы облегчить отладку, DevTools теперь поддерживает вычисление выражений с помощью частных членов класса. обновления раньше релиза</h2><p>Рассмотрите возможность использования Chrome <a href="https://www.google.com/chrome/canary/" rel="noopener noreferrer nofollow">Canary</a>, <a разработки по умолчанию. </p><p>Эти каналы предварительного просмотра предоставляют вам доступ к новейшим функциям DevTools, тестируют передовые API веб-платформы и обнаруживают проблемы на вашем
