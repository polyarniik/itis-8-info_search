<br/>
Пару лет назад <a href="https://habr.com/ru/company/selectel/blog/551348/">мы писали про ноутбук MNT Reform</a>. Он предназначен для пользователей, которым крайне важна сохранность персональных данных. Это устройство было даже названо «ноутбуком для параноиков», хотя, конечно, это не официальное название. <br/>
<br/>
Ну а теперь разработчики этого девайса выпустили новое устройство, только с диагональю экрана в 7 дюймов. К слову, слоган компании: «цифровое будущее будет модульным и open-source». Эта же команда поддерживает «право на ремонт», идеей, с которой полностью совместимы, если так можно выразиться, все девайсы MNT. Теперь к ним добавился и этот небольшой лэптоп, который получил название MNT Pocket Reform. Подробности — под катом. <br/>
<a name="habracut"></a><br/>
<font color="#EB4247"><h3>Что собой представляет этот девайс? </h3></font><br/>
По словам разработчиков, MNT Reform представляет собой:<br/>
<br/>
<ul>
<li>Почти полностью открытый ноутбук, который можно модифицировать как душе угодно, заменять сломанные модули или менять устаревшие компоненты на более производительные/функциональные.</li>
<li>Систему с защитой персональных данных. В ноутбуке нет камер, микрофонов, WiFi можно отключать. При необходимости персональные данные можно хранить на SD-карте. Есть и шифрование жесткого диска с LUKS.</li>
<li>Печатную машинку, которую можно использовать для офисных целей. Любые офисные программы, браузер и т.п. работают без проблем.</li>
<li>Промышленную систему. Благодаря чипсету ноутбук можно использовать в качестве промышленной «рабочей лошадки».</li>
<li>Платформу для разработки ARM 64-bit ПО. Ну, здесь и так все понятно.</li>
<li>Обучающую систему. MNT Reform можно использовать в качестве базы для обучения принципам работы железа, ПО, защиты информации и т.п. Подходит и детям, и взрослым.</li>
</ul><br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/dv/8w/yp/dv8wypqsnvz7kyquhgsus6vhov0.jpeg" src="https://habrastorage.org/r/w780q1/webt/dv/8w/yp/dv8wypqsnvz7kyquhgsus6vhov0.jpeg"/><br/>
<br/>
<font color="#EB4247"><h3>А что с характеристиками? </h3></font><br/>
В целом, прорывом в производительности этот ноутбук назвать нельзя. Вот основные технические спецификации: <br/>
<br/>
<ul>
<li>CPU: NXP i.MX8M Plus с 4 ядрами ARM Cortex-A53 (1.8 GHz), 1x Cortex-M4F ядром. CPU и RAM размещены на съемном модуле размером с SO-DIMM.</li>
<li>RAM: 4 или 8 GB LPDDR4.</li>
<li>GPU: Vivante GC7000UL GPU.</li>
<li>Display: Full HD (1920x1080 pixels) 7" IPS eDP display driven via MIPI-DSI. Micro HDMI с выводом изображения в 4К качестве. </li>
<li>2 USB-C порта, один из них поддерживает зарядку. </li>
<li>MicroSD слот.</li>
<li>ix Industrial Ethernet порт с опциональным адаптером для RJ45.</li>
<li>WWAN слот для 4G/5G/LTE.</li>
<li>Слот для micro SIM карты. </li>
<li>Беспроводный модуль связи с поддержкой WiFi 802.11ac и Bluetooth 5.0.</li>
<li>Механическая клавиатура с подсветкой. </li>
<li>Размер: 20 × 12.6 × 4.5 см.</li>
</ul><br/>
Кроме базовой конфигурации, доступны еще и опциональные: <br/>
<br/>
<ul>
<li>NXP Layerscape LS1028A (2× ARM Cortex-A72, 8 или 16 ГБ DDR4, Vivante GC7000UL GPU)</li>
<li>Raspberry Pi CM4 (с адаптером, 4× ARM Cortex-A72, 8GB DDR4, VideoCore GPU)</li>
<li>Pine SOQuartz RK3566 (с адаптером, 4× ARM Cortex-A55, 8GB DDR4, Mali G52 GPU)</li>
<li>FPGA AMD/Xilinx Kintex-7 (для промышленного использования, RISC-V SoC возможный вариант)</li>
<li>Дополнительные модули для устройства. </li>
</ul><br/>
<img data-src="https://habrastorage.org/webt/nb/_t/4v/nb_t4vaauzjoyhalgs9pmdwx1zu.png" src="https://habrastorage.org/r/w1560/webt/nb/_t/4v/nb_t4vaauzjoyhalgs9pmdwx1zu.png"/><br/>
<br/>
Как и говорилось выше, производительной эту систему назвать нельзя. Разработчики утверждают, что она по этому показателю идет примерно вровень с Raspberry Pi 4. Для серфинга, разного рода промышленных приложений, просмотра видео (при необходимости) этого вполне хватит. В более-менее продвинутые игры же играть не получится, да и сам девайс сделан вовсе не для этой цели. <br/>
<br/>
Основное достоинство материнской платы девайса — возможность подключения системы на модуле (SoM). Как и говорилось выше, модуль с процессором, видеочипом и ЗУ вставляется в материнку как SO-DIMM.<br/>
<br/>
<a href="https://selectel.ru/services/dedicated/?utm_source=habr.com&amp;utm_medium=referral&amp;utm_campaign=dedicated_article_dedicateds_180223_MNT"><div style="text-align:center;"><img data-src="https://habrastorage.org/webt/5_/4y/nr/5_4ynrojdbgluj59gv4aqahnook.png" src="https://habrastorage.org/r/w1560/webt/5_/4y/nr/5_4ynrojdbgluj59gv4aqahnook.png"/></div></a><br/>
<font color="#EB4247"><h3>А для чего вообще его можно использовать? </h3></font><br/>
<img data-blurred="true" data-src="https://habrastorage.org/webt/2g/fh/ls/2gfhlsh91_kluy2ayfqrhfz_ggc.jpeg" src="https://habrastorage.org/r/w780q1/webt/2g/fh/ls/2gfhlsh91_kluy2ayfqrhfz_ggc.jpeg"/><br/>
<br/>
Разработчики утверждают, что устройство можно применять, когда требуется быстро набрать текст в «полевых» условиях, где обычный ноутбук либо недоступен, либо его по какой-то причине использовать не совсем удобно.<br/>
<br/>
Коммуникации — еще она сильная сторона устройства. Его можно использовать для работы электронной почтой или чатами в Mastodon, Telegram, Twitter, IRC или Matrix. Здесь большой плюс — в наличии модулей для мобильной связи, которая позволяет выходить в сеть практически отовсюду. <br/>
<br/>
Системным администраторам такой девайс может очень сильно пригодиться — ведь, благодаря наличию все того же беспроводного модуля связи, устройство отлично подходит для удаленной работы. <br/>
<br/>
Инженеры могут применять девайс на производстве. Насколько известно, он не особо защищено от пыли, воды и всего такого прочего, но небольшой девайс удобнее во многих случаях, чем устройства привычного размера. <br/>
<br/>
<font color="#EB4247"><h3>А что насчет размера — удобно ли работать с таким крохотным девайсом? </h3></font><br/>
Тестировщики, кому уже удалось поработать с устройством, отмечают, что в первые часы работы скорость печати или вообще взаимодействия с этим девайсом замедляется. Но потом, благодаря продуманной клавиатуре (выглядит она немного непривычно), скорость печати восстанавливается. Ну а ко всему остальному пользователь постепенно адаптируется. <br/>
<br/>
Небольшой экран потребляет относительно немного энергии. Разработчики установили в девайс литиевые батареи с общей емкостью в 8000 мА*ч, чего в обычном режиме хватает на несколько часов работы. <br/>
